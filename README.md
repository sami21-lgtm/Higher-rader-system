# 🛰️ Futuristic Neon Radar System (Ver. 1, 2 & 3)

A premium, high-tech tactical radar simulation web interface. This project features a solid pitch-black background with dynamic neon lighting effects, transitions, and an ultra-attractive glowing developer credit section. Fully responsive and optimized for both desktop and mobile displays.

---

## 🚀 Features

* **Pure Black Theme:** Absolute dark background (`#000000`) for maximum visual contrast and power-saving on OLED/AMOLED mobile screens.
* **Dynamic 3-State Lighting Controller:** Smooth slider switch to toggle between three classic cyberpunk neon views:
  * 🔵 **Ver. 1:** Cyber Cyan / Neon Blue
  * 🔴 **Ver. 2:** Crimson Red / Danger Zone
  * 🟢 **Ver. 3:** Tactical Matrix Green (Default View)
* **Responsive Scaling:** Uses fluid viewport units (`vw`) ensuring the radar sweeps and rings scale beautifully to fit any mobile phone screen size.
* **Premium Glowing Credit Panel:** An integrated developer block with multi-layered text-shadows and breathing pulse animation that dynamically matches the selected version's color scheme.
